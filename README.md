# ClawGuru – User Magnet Edition

Deploy-ready for **Vercel** and **Netlify**.

## Local
```bash
npm install
npm run dev
```

## Netlify
- Build: `npm run build`
- Publish: `.next`
- Plugin: `@netlify/plugin-nextjs` (already configured in netlify.toml)

## Vercel
Import repository – default settings.

## Notes
- Next.js pinned to 14.2.21 (Netlify CVE gate safe).
- React pinned to 18.2.
